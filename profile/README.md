## FluxBalancer

**FluxBalancer** — это исследовательский проект и прототип интеллектуального балансировщика нагрузки для распределённых сервисов, разработанный в рамках выпускной квалификационной работы.

Проект реализует **адаптивную маршрутизацию HTTP-запросов** на основе многокритериального анализа (MCDM). Для выбора оптимального узла используются метрики текущего состояния системы (CPU, память, сеть, задержка), автоматически собираемые из Docker-контейнеров, а также формальные методы принятия решений, такие как **TOPSIS, SAW, ELECTRE, AIRM и линейная скаляризация**.

Ключевые особенности:

* сбор и агрегация метрик узлов в реальном времени;
* поддержка нескольких алгоритмов многокритериального выбора;
* автоматическое вычисление весов критериев (энтропийный метод);
* проксирование запросов с учётом SLA и дедлайнов;
* модульная архитектура с чётким разделением domain / application / adapters;
* асинхронная реализация на FastAPI и aiohttp.

Проект предназначен для исследования и сравнения стратегий балансировки нагрузки в условиях динамически изменяющихся ресурсов и может служить основой для дальнейших экспериментов и расширений.
